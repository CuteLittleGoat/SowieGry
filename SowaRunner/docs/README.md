# SowaRunner — instrukcja obsługi

### Jak uruchomić
1. Otwórz plik `SowaRunner/index.html` w przeglądarce.
2. Na ekranie startowym wybierz poziom trudności (1 / 2 / 3 lub kliknięcie).
3. Naciśnij **Spację** lub kliknij przycisk Start, aby rozpocząć grę.

### Sterowanie
- **Spacja / klik / tap** — start gry oraz skok.
- **Podwójny skok** — dwa szybkie skoki w powietrzu.
- **1 / 2 / 3** — wybór poziomu trudności na ekranie startowym.
- **R** — powrót do ekranu startowego w trakcie gry.

### Dźwięk
- Pierwsze kliknięcie/tap włącza dźwięk (wymóg przeglądarek).
