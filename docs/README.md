# SowieGry — instrukcja obsługi

### Jak uruchomić
1. Otwórz plik `index.html` w głównym katalogu repozytorium w przeglądarce.
2. Na ekranie startowym kliknij kartę wybranej gry (`SowaRunner` lub `SowaJumper`).

### Sterowanie na ekranie startowym
- Kliknij wybraną kartę gry, aby przejść do folderu gry.

### Wskazówki
- Najlepiej uruchamiać gry w trybie pełnoekranowym przeglądarki.
- Jeśli gra nie ładuje się poprawnie, odśwież stronę.
